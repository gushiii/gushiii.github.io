JDK安装和配置

下载JDK

在Oracle官网下载


**单击红框中标出的download按钮下载**


**注意要先选择Accept License Agreement 之后下载Windows对应的jdk(JDK10 不支持X86的系统)**


**全部默认，一路下一步**

配置JDK

**右键我的电脑，单击属性，选择高级系统设置**


**选择环境变量**


**在系统变量下单击新建，输入变量名“JAVA_HOME，输入变量值位JDK的安装路径”**


**选中系统变量中的path变量，单击编辑按钮在变量中添加**


**在系统变量中新建CLASSPATH变量，变量值为 . **


**配置完成后在命令行中检验是否配置成功**
**具体步骤：**
    1. 键盘输入win + r
    2. 输入 cmd 单击运行
    3. 在命令行中输入 javac 然后回车，出现以下结果则为配置成功
    4. 若出现"javac不是内部命令或外部命令，也不是可执行的程序或批处理文件"，检查配置过程是否有错
